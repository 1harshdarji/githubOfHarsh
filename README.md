# Harsh Darji Portfolio

This is the personal portfolio website for Harsh Darji, a Full-Stack Engineer and Open Source Enthusiast.

## About

The `index.html` file contains a modern, responsive portfolio website featuring:

- **Dark Theme Design**: Sleek black background with white text and subtle gradients
- **Starfield Animation**: Animated background with falling stars and shooting stars using HTML5 Canvas
- **Project Showcase**: Grid layout displaying various engineering projects with images and technology tags
- **Responsive Layout**: Mobile-friendly design with adaptive typography and grid system
- **Smooth Interactions**: Hover effects and transitions for enhanced user experience

## Features

### Visual Elements
- Animated starfield background with 500+ stars
- Shooting star effects every 5 seconds
- Glassmorphism card design with backdrop blur
- Custom scrollbar styling

### Content Sections
- **Header**: Name, title, and subtitle introduction
- **Projects Grid**: 3-column responsive grid showcasing engineering projects
- **Footer**: Copyright information with dynamic year

### Project Showcased
- C++, Python, JavaScript, TypeScript
- React, Node.js, Docker, Kubernetes
- AWS, TensorFlow, MQTT, TimescaleDB
- Embedded Systems, Real-Time OS, Data Science
